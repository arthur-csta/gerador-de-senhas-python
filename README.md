# Gerador de Senhas em Python

Projeto simples desenvolvido para praticar lógica de programação em Python.

## Funcionalidades
- Escolha da quantidade de senhas
- Escolha do tamanho das senhas
- Inclusão de letras
- Inclusão de números
- Inclusão de símbolos

## Tecnologias
Python 3
